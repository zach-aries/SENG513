Eggstinction

Mobile Game

Description

Steve Morris and I created Eggstinction for our final project during our time at SAIT in the NMPD program.

The goal was to create a fully functional mobile puzzle game for tablets. Steve did all the graphic design and I coded the project.

You must enable flash in order to play.

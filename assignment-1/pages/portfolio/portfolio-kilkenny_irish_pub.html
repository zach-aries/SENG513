Kilkenny's Irish Pub

Ordering Kiosk

Description

This project was created in CPSC 481 - Human Computer Interaction I, with the following synopsis:

Kilkenny's Irish pub has hired us to create an automatic ordering system for their customers to streamline the ordering and payment processes. Servers' jobs would become easier as they would face a reduced amount of responsibilities and customers would experience quicker service. The pub itself would find lower long-term costs as they would require less servers to cover the floor.
